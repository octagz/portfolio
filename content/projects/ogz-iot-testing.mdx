# Introduction

At digitalstrom AG in Zurich, Switzerland, I worked on IoT firmware for smart home automation devices and led the test automation initiative. The company's devices — smart switches, dimmers, and controllers — were deployed in over 10,000 homes across Europe, enabling residents to control lighting, heating, and appliances through a unified system.

&nbsp;

# The Challenge

Smart home devices operate under strict constraints: limited memory, real-time responsiveness requirements, and the expectation of near-perfect reliability. A firmware bug in a deployed device can mean a homeowner's lights don't turn on — or worse, a safety issue. The existing manual testing process was slow, error-prone, and couldn't keep pace with the development cycle.

&nbsp;

# Testing Framework Architecture

I designed and implemented an automated testing framework to ensure firmware quality before deployment:

- **Python test suites**: Comprehensive test coverage for firmware behavior, communication protocols, and device interaction scenarios
- **CI/CD integration**: Jenkins and GitlabCI pipelines that ran the full test suite on every commit, catching regressions early
- **Hardware-in-the-loop testing**: Test rigs that exercised actual device hardware, not just software simulations, ensuring real-world behavior matched specifications
- **Performance benchmarking**: Automated energy efficiency and response time measurements to track device performance across firmware versions

&nbsp;

# Firmware Development

Beyond testing, I contributed to the C++ firmware running on the devices themselves:

- Optimized memory usage and execution paths to improve performance on resource-constrained hardware
- Implemented communication protocol features for device-to-device and device-to-cloud interaction
- Improved energy efficiency in deployed units through firmware optimizations

&nbsp;

# Results

- **Boosted reliability and deployment speed by 50%** through automated testing and CI/CD
- **Improved performance and energy efficiency** in 10K+ deployed smart home units
- **Reduced regression incidents** by catching issues before they reached production firmware
- **Established testing culture** that the team continued to build on after my tenure
