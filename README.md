## Environment variables

Create a `.env.local` with the following keys before running the app locally or deploying:

- `NEXT_PUBLIC_GA_ID` – Google Analytics 4 Measurement ID (e.g. `G-XXXXXXX`)
- `NEXT_PUBLIC_BEAM_TOKEN` – Optional Beam Analytics token for the Beam widget
